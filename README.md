## 📁 Project Files Included
- `index.html`: The main web page containing the structure and text of the website.
- `github_picture.png`: The profile/screenshot image used in the layout.

## 🛠️ Technologies & Elements Used
- **HTML5 Structure:** Used headings (`<h1>`, `<h2>`), paragraphs (`<p>`), bold text (`<b>`), and horizontal dividers (`<hr>`).
- **Images & Links:** Integrated profile images via `<img>` and hyperlinked external platforms using `<a>`.
- **Lists:** Structured core computing tools and skills utilizing unordered lists (`<ul>` & `<li>`).
- **Inline Styling:** Adjusted visibility and text layout using the `style` attribute (e.g., `font-size` and image `width`).
